# Simon Game

A classic memory game built with HTML, CSS, and Vanilla JavaScript. Test your memory by repeating a sequence of increasingly complex color patterns!

## 🎮 How to Play

1. Press any key on your keyboard to start the game.
2. The game will show you a sequence by flashing one of the colors.
3. Repeat the sequence by clicking the correct color block.
4. If you get it right, the level increases, and one more color is added to the sequence.
5. If you click the wrong color, the game is over and your score is displayed.
6. Press any key to restart and try again!

## ✨ Features

- **Progressive Difficulty:** The sequence gets longer and harder to remember with each level.
- **High Score Tracking:** Your highest level reached is saved during your current session.
- **Visual Feedback:** Color blocks flash both when the game demonstrates the sequence and when you click them. Also includes a red flash effect when the game is over.

## 🛠️ Technologies Used

- **HTML5:** Structure of the game interface.
- **CSS3:** Styling, layout using flexbox/grid, and visual effects for buttons.
- **JavaScript:** Game logic, event handling, sequence generation, and DOM manipulation.

## 🚀 How to Run Locally

Since this is a front-end only project, no installation or build steps are required.

1. Clone or download the repository to your local machine.
2. Navigate to the project folder.
3. Open `index.html` in your favorite web browser (Chrome, Firefox, Safari, Edge, etc.).

Alternatively, if you use VS Code, you can open the project folder and use the "Live Server" extension to run it.

## 📁 File Structure

- `index.html` - The main HTML file containing the structure.
- `style.css` - Styles application and layout.
- `script.js` - Contains all the interactivity and game logic.
